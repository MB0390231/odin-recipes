Description

Steps for creating html boilerplate

Ingredients

  • DOCTYPE
  • html lang
  • title
  • head
  • body
  1. Declare the docktype in the first line
  2. declare type (html) and lang
  3. create start of head
  4. create character set through meta charset within head
  5. create title within head
  6. end head
  7. create and end body
